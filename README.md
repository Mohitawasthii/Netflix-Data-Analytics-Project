## 📌 Project Overview

This project performs an end-to-end **Exploratory Data Analysis (EDA)** on the **Netflix Movies & TV Shows Dataset** to uncover insights into content distribution, genres, ratings, release trends, countries, and durations. The project demonstrates a complete data analytics workflow, transforming raw streaming data into meaningful business insights through visualization and statistical analysis.

## 🎯 Project Objectives

- Analyze Netflix movies and TV shows.
- Perform data cleaning and preprocessing.
- Explore content trends across genres, countries, and years.
- Study ratings, durations, and release patterns.
- Create informative visualizations.
- Generate actionable business insights.

## 📂 Dataset Features
